# Blueshot

Blueshot is a web platform designed for teams to easily schedule and manage their meetings and events. The core idea is to provide a simple, fast, and collaborative space for organizing team activities, with a focus on saving time and reducing friction.

## Project Overview

- **Current Features:**
  - **Calendar View:** Teams can view and manage upcoming events in a shared calendar.
  - **Event Scheduling:** Create, edit, and track meetings or events for your group.
  - **User Groups:** Organize users into groups for targeted event management.
  - **Event Participation Tracking:** Track which users have acknowledged or responded to event invitations.
  - **Permissions:** Manage who can create or edit events within groups.
  - **Authentication:** Secure sign-in and registration for users.

- **Planned Features:**
  - **Team Chat:** Real-time chat for team communication.
  - **Video Calls:** Integrated video conferencing for scheduled events.
  - **More Collaboration Tools:** Additional features to make team coordination even easier.

## Vision

Blueshot aims to be the go-to platform for teams who want to spend less time organizing and more time collaborating. While the current version focuses on calendar and event management, future updates will bring richer communication and collaboration features.
